# py-pyautogui

PyAutoGUI を使った画面自動操作スクリプト。画面上から指定した画像（`assets/target.jpg`）を探し出し、見つかったら自動でクリックする単純な GUI 自動化ツールです。

## 必要環境

- Python 3.14（`.python-version` で指定）
- パッケージ管理: [uv](https://docs.astral.sh/uv/)

## 依存パッケージ

| パッケージ | 役割 |
| --- | --- |
| `pyautogui` | マウス操作・画面キャプチャ・画像検索 |
| `opencv-python` | `confidence`（曖昧一致）による画像マッチの精度向上 |
| `pillow` | 画面キャプチャの内部処理（`pyscreeze` が要求） |

> **補足**: `pyscreeze` は Python 3.12 以降向けに Pillow 依存をメタデータ宣言していないため、Pillow を明示的に追加しています。

## セットアップ

```powershell
# 依存をインストール（初回のみ）
uv sync
```

## 使い方

1. 画面上で探したい対象のスクリーンショットを `assets/target.jpg` として配置する
2. スクリプトを実行する

```powershell
uv run main.py
```

実行すると、画面内に `target.jpg` と一致する箇所があれば、その中心を自動でクリックし続けます。

### 動作の流れ

スクリプトは以下を無限ループで繰り返します。

1. `locateCenterOnScreen` で画面内から `target.jpg` を探す
2. 見つかればその中心をクリック（見つからなければメッセージを表示）
3. 4 秒待機する
4. マウスを画面右下から 100px 内側へ移動する
5. 1 へ戻る

## チューニング

`main.py` 先頭の `CONFIDENCE` で画像マッチの厳しさを調整できます（`1.0` = 完全一致）。

| 症状 | 対処 |
| --- | --- |
| 検出漏れ（表示されているのに見つからない） | `CONFIDENCE` を `0.7` 程度へ下げる |
| 誤検出（別の場所をクリックする） | `CONFIDENCE` を `0.85`〜`0.9` へ上げる |
| それでも不安定 | `assets/target.jpg` を PNG で撮り直す（JPEG はノイズが乗るため） |

## 緊急停止（フェイルセーフ）

マウスを画面の四隅（特に左上）へ勢いよく動かすと、PyAutoGUI のフェイルセーフが作動してスクリプトが停止します。暴走時の停止手段として覚えておいてください。

## 注意

- `assets/` は `.gitignore` 対象です。`target.jpg` はローカル専用で、リポジトリにはコミットされません。
